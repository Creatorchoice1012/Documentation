Clone the Repository

git clone https://github.com/Creatorchoice1012/AccountsModule.git

Navigate into the project directory and install dependencies:

cd account-module
npm install

Usage

To integrate the account module into your project, import the necessary components and set up the module as shown below.

Sign-Up Component Example


import React from 'react';
import { SignUpComponent } from 'account-module';

const App = () => {
  return (
    <SignUpComponent
      username={true}
      emailField={true}
      passwordField={true}
      confirmPasswordField={true}
      rememberMeCheckbox={true}
      signInScreen="LoginScreen"
      verifyScreen="VerifyScreen"
    />
  );
};

export default App;

Sign-In Component Example


import React from 'react';
import { SignInComponent } from 'account-module';

const App = () => {
  return (
    <SignInComponent
      username={true}
      passwordField={true}
      signUpScreen="SignUpScreen"
      signInProviders={['google', 'facebook','whatsapp']}
      rememberMeCheckbox={true}
    />
  );
};

export default App;

Props

Prop Type Description Default
username boolean Display username input field true
emailField boolean Display email input field true
mobileNumber boolean Display mobile number input field true
passwordField boolean Display password input field true
confirmPasswordField boolean Display confirm password input field true
rememberMeCheckbox boolean Display 'Remember Me' checkbox false
signUpButton boolean Display sign-up button true
signInScreen string Navigation screen for sign-in button LoginScreen
verifyScreen string Navigation screen after OTP verification VerifyScreen

Customization

You can customize the appearance and behavior of the form by passing props like username, emailField, and more.


<SignUpComponent
  username={false} // Hides the username field
  mobileNumber={true} // Shows the mobile number field
  rememberMeCheckbox={true} // Shows 'Remember Me' checkbox
/>
